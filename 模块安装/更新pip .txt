方法：在windows启动CMD窗口,输入python -m pip install --upgrade pip  点击回车即可。
4.pip更新完成以后，同样的cmd窗口，输入pip install pillow   回车即可。
5.无须更多操作！！！！！

注：
在第三步的时候很可能报错，很大概率事件是点击回车之后出现
collect pip
进度条开始走，然后卡住，然后弹出很多红字----更新失败
楼主当时重复了第三步很多次，每次都失败，而且每次进度条卡住的位置不同，如果出现类似的情况！！！是因为网络不好，和国外的服务器连接超时！

解决方法：翻墙，搞个VPN，楼主推荐一下，这里真的不是打广告，天行vpn，推荐的原因只有一个，注册账号给一个小时免费的试用，打开VPN, 选一个快一些的线路。然后重新操作上述的第三第四步，分分钟就更新完成。
